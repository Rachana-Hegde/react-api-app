# React API Fetch App

A simple React application that fetches data from a public API and displays it in a clean and responsive user interface.

## Features

- Fetches data from a public API
- Displays data in a card-based UI
- Shows loading state while fetching data
- Handles error states if the API fails
- Fully responsive design

## Tech Stack

- React
- Vite
- JavaScript
- CSS

## API Used

This project uses the free public API:

https://jsonplaceholder.typicode.com/posts

## Installation

1. Clone the repository

```bash
git clone https://github.com/Rachana-Hegde/react-api-app.git
```

2. Navigate to the project folder

```bash
cd react-api-app
```

3. Install dependencies

```bash
npm install
```

4. Run the development server

```bash
npm run dev
```

5. Open in browser

```
http://localhost:5173
```
